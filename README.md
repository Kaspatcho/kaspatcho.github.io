
# Painel EMTU
Este projeto simula uma placa de ônibus inteligente que mostra aos usuários o tempo estimado de chegada
de cada ônibus na parada. O objetivo é fornecer informações em tempo real para que os usuários
possam planejar melhor suas viagens e ter uma maior qualidade de vida.

## Tecnologias utilizadas
- Node.js como servidor backend
- JavaScript como linguagem de programação
- HTML, CSS e Bootstrap para a interface do usuário
- API da EMTU para obter informações em tempo real sobre os ônibus

## Funcionalidades
- Exibe o tempo estimado de chegada de cada ônibus na parada
- Mostra informações sobre a linha, destino e tempo de viagem
- Atualiza automaticamente as informações a cada 1 minuto

## Disponibilidade
O site está disponibilizado em [Painel EMTU](https://painel-emtu.vercel.app/).

## Como funciona
O site utiliza a API da EMTU para obter informações em tempo real sobre os ônibus.
Essas informações são processadas e exibidas na interface do usuário. O site atualiza automaticamente as
informações a cada 1 minuto para garantir que os usuários tenham acesso às informações mais recentes.

## Como iniciar o projeto em sua máquina local
1. Certifique-se de ter o [Node.js](https://nodejs.org) instalado em sua máquina.

2. Clone o repositório do projeto em sua máquina local:
```bash
git clone https://github.com/Kaspatcho/kaspatcho.github.io.git
```

3. Navegue até o diretório do projeto:
```bash
cd kaspatcho.github.io
```

4. Instale as dependências do projeto:
```bash
npm install
```

5. Inicie o servidor:
```bash
npm start
```

6. Abra o navegador e acesse o endereço http://localhost:3000 para ver o projeto em execução.
